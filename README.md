# Forest Three Nights

iPadのタップ操作を前提にした、ブラウザで動く2Dサバイバルゲームの初期プロトタイプです。

## Play

`index.html` をブラウザで開くと動きます。GitHub Pagesではリポジトリのルートを公開すればそのまま配信できます。

## Current Rules

- タップした場所へ移動
- 近くの木材、ベリー、石を `Gather` で採取
- `Eat` でベリーを食べて空腹を回復
- 焚き火の近くで `Fire` を押すと木材を消費して火を強める
- 夜は敵が現れ、寒さと空腹も厳しくなる
- 3日目の夜明けまで生き残れば勝利
