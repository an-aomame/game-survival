# 森の三日間

iPadのタップ操作を前提にした、ブラウザで動く2Dサバイバルゲームの初期プロトタイプです。

## 遊び方

`index.html` をブラウザで開くと動きます。GitHub Pagesではリポジトリのルートを公開すればそのまま配信できます。
タイトル画面のバージョン番号で表示中の版を確認できます。更新が反映されないときは `最新版に更新` を押すと、キャッシュを避けて再読み込みします。
iPad Safariでページ自体が拡大されないよう、ピンチやダブルタップによるブラウザズームをゲーム画面では抑制しています。
タイトル画面とゲーム中の `?` から、博士がルールとコツを説明するヘルプを開けます。

## 現在のルール

- タップした場所へ移動
- 近くの木材、ベリー、石を `採取` で集める
- `食べる` でベリーを食べて空腹を回復
- 焚き火の近くで `焚き火` を押すと木材を消費して火を強める
- 焚き火の近くで `囲い` を押すと石3個を消費して安全地帯を広げる
- 焚き火の外側の点線は暖かさ、内側の実線は敵よけの目安
- 夜は敵が現れ、寒さと空腹も厳しくなる
- 10日目の夜明けまで生き残れば勝利
